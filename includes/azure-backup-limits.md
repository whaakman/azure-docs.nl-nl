De volgende limieten gelden voor Azure Backup.

| **Limiet** | **Standaard** |
| --- | --- |
| Servers of machines dat kunnen worden geregistreerd in een kluis. | Windows Server/Windows-Client/System Center Data Protection Manager: 50. <br/><br/> IaaS-VM's: 1,000.  |
| Grootte van een gegevensbron in de opslag van de kluis. |Maximaal 54,400-GB. De limiet niet van toepassing op IaaS VM backup. |
| Back-upkluizen in een Azure-abonnement. |500 kluizen per regio. |
| Dagelijkse back-ups plannen. |Windows Server/Client: Drie per dag.<br/> System Center DPM: Twee per dag. <br/> IaaS-VM's: Eenmaal per dag.  |
| Gegevensschijven die zijn gekoppeld aan een Azure-VM voor back-up. | 16 |
| Afzonderlijke gegevensschijf gekoppeld aan Azure-VM voor back-up.| 4095 GB|

[Azure App Service](/azure/app-service/overview) kunt u bouwen en hosten van code. Deze code kan worden in de vorm van web-apps, mobiele back-ends en RESTful-API's. U kunt u code met behulp van de programmeertaal van uw keuze zonder dat infrastructuur worden beheerd met behulp van App Service, ontwikkelen. App Service ondersteunt zowel Windows als Linux. Geautomatiseerde implementaties van een willekeurige Git-repo worden ondersteund, met inbegrip van GitHub en DevOps met Azure.

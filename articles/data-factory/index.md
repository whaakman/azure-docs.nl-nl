# <a name="azure-data-factory-documentation"></a>Documentatie voor Azure Data Factory

Azure Data Factory is een cloudgebaseerde service voor gegevensintegratie waarmee de verplaatsing en transformatie van gegevens worden geautomatiseerd. Met de Data Factory-service maakt u oplossingen voor gegevensintegratie waarmee u gegevens uit verschillende gegevensarchieven kunt opnemen, de gegevens kunt transformeren en verwerken, en de resulterende gegevens naar de gegevensarchieven kunt publiceren.

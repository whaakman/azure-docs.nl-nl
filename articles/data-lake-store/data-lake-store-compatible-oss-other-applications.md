# <a name="open-source-big-data-applications-that-work-with-azure-data-lake-storage-gen1"></a>Open Source Big Data-toepassingen die met Azure Data Lake Storage Gen1 werken

[!INCLUDE [data-lake-storage-gen1-rename-note.md](../../includes/data-lake-storage-gen1-rename-note.md)]

Voor de toepassingen in de onderstaande tabel zijn alleen de versies die beschikbaar zijn met de vermelde distributiepunten worden ondersteund. Zie voor informatie over welke versies van deze toepassingen beschikbaar met HDInsight zijn, [versiebeheer van HDInsight-onderdeel](../hdinsight/hdinsight-component-versioning.md).

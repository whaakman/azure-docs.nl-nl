# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Aan de slag met de herkenning van U-SQL

## <a name="overview"></a>Overzicht
Cognitieve functionaliteit voor U-SQL kunnen ontwikkelaars put intelligence gebruiken in hun big data-programma. 

De volgende voorbeelden met behulp van cognitieve mogelijkheden zijn beschikbaar:
* Installatiekopieën: [Detecteer gezichten](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Installatiekopieën: [Detecteren van emoties](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Installatiekopieën: [Detecteren van objecten (tags)](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Installatiekopieën: [OCR (Optische tekenherkenning)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Tekst: [Sleuteltermextractie & Sentimentanalyse](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)

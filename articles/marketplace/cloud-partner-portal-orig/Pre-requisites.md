# <a name="prerequisites-for-application-curation"></a>Vereisten voor de toepassing Curatie

Microsoft vereist specifieke beoordelingen om te valideren dat de volgende vereisten wordt voldaan:

- Een partner\'s aangepaste code voldoet aan de richtlijnen van Microsoft.
- Een oplossingspakket Microsoft Dynamics Lifecycle Services (LCS) kan worden geïmplementeerd.
- Transacties kunnen worden uitgevoerd.

Op dit moment partners moeten aantonen dat aan deze vereisten is voldaan door testimplementaties uitvoert en de resultaten vervolgens te delen met Microsoft. Er is geen code geïmplementeerd op de omgeving van een klant die Microsoft-berichten\'t gevalideerd. Partners moeten de volgende curatie-artefacten en tests uitvoeren:

- Rapport van de analyse van code (CAR)
- Zakelijke proces samenstellen (BPM) en scripts testen
- Naam van het project en beschrijving
- Back-up van Business-database
- Gegevenspakketten en proces gegevenspakketten (aanvraag)
- Methodologie
- Binaire bestanden (optioneel)
- Implementeerbare pakketten
- Modellen (code en tests)
- Marketing-inhoud

Zie voor meer informatie over technische en vereisten [valideren van toepassingen voor Finance and Operations](https://docs.microsoft.com/dynamics365/operations/dev-itpro/lcs-solutions/app-validation-lcs-solutions).  Als u hebt nog niet gedaan, Bekijk ook de [Office 365, Dynamics 365, PowerApps en Power BI bieden Publicatiehandleiding voor](../appsource-offer-publishing-guide.md).

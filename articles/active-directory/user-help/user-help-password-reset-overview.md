# <a name="reset-password-overview"></a>Overzicht van wachtwoord opnieuw instellen

>[!Important]
>Deze inhoud is bedoeld voor gebruikers. Als u een beheerder bent, kunt u meer informatie vinden over hoe u uw Azure Active Directory-omgeving (Azure AD) instelt en beheert in de [documentatie voor Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

Als u uw wachtwoord bent vergeten, nooit een wachtwoord van bedrijfsondersteuning hebt ontvangen, of uw account is vergrendeld, kunt u uw beveiligingsgegevens en mobiele apparaat gebruiken om uw wachtwoord opnieuw in te stellen.

|------|------------|

|[Uw wachtwoord opnieuw instellen](user-help-reset-password.md)| Beschrijft hoe u uw eigen wachtwoord opnieuw kunt instellen.|

## <a name="one-identity-authentication-services"></a>Verificatieservices één identiteit

![Één identiteit](./media/compatible-software-icons/OneIdentity.png)

Een identiteit verificatieservices kunt Unix, Linux en Mac OS X-systemen om de toegang, verificatie en autorisatie van de bestaande Active Directory (AD)-infrastructuur van een organisatie gebruiken. Verificatieservices biedt nu ondersteuning voor Azure Active Directory Domain Services inschakelen niet-Windows-resources gebruikmaken van dezelfde generatie-platform die al gebruikmaken van uw bestaande SaaS-oplossingen.

## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux is van de hele wereld toonaangevende enterprise Linux-platform gebouwd om te voldoen aan de behoeften van de moderne onderneming van vandaag. Red Hat Enterprise Linux is de beste keuze voor workloads van enterprise Linux virtuele machine (VM) in Microsoft Azure. Red Hat Enterprise Linux is een open, betrouwbaar en veilig platform is ontworpen voor klanten die flexibiliteit van de implementatie willen voor hun bedrijfskritische workloads, van het datacenter naar de Azure-cloud - ondersteund door nauw geïntegreerd, ondersteuning van bedrijfskwaliteit Red Hat en Microsoft.

# <a name="using-the-deep-learning-virtual-machine"></a>Met behulp van de Deep Learning virtuele Machine

Nadat u uw Deep Learning Virtual Machine (DLVM) hebt ingericht, kunt u beginnen het bouwen van deep neural network-modellen vertrekken AI-toepassingen in domeinen, zoals computer-vision en language understanding. 

Er zijn een aantal hulpprogramma's op de Deep Learning VM voor AI. De [pagina van frameworks voor Deep Learning en AI](dsvm-deep-learning-ai-frameworks.md) bevat details over het gebruik van deze hulpprogramma's. 

## <a name="deep-learning-tutorials-and-walkthroughs"></a>Deep Learning-zelfstudies en -scenario's

Naast de voorbeelden framework op basis van is een set van uitgebreide scenario's ook beschikbaar die zijn gevalideerd op de DLVM. Deze scenario's kunt u snel uw ontwikkeling van deep learning-toepassingen in domeinen, zoals afbeeldingen en tekst/language understanding. Meer zelfstudies voor end-to-end in verschillende domeinen en -technologie blijft worden toegevoegd.   


- [Met neurale netwerken in verschillende frameworks](https://github.com/ilkarman/DeepLearningFrameworks): Een uitgebreide procedure die laat zien hoe u code van een framework migreren naar een andere. Ook zien hoe om te vergelijken model en de prestaties van de tijd van frameworks uitvoeren. 

- [Handleiding voor het bouwen van een end-to-end oplossing voor het detecteren van producten in afbeeldingen](https://github.com/Azure/cortana-intelligence-product-detection-from-images): Detectie van de installatiekopie is een techniek die kan vinden en classificeren van objecten in afbeeldingen. Deze technologie is de mogelijkheid om enorme voordelen in veel echte leven domeinen. Bijvoorbeeld, kunnen detailhandelaren deze techniek gebruiken om te bepalen welk product een klant is opgehaald uit de plank. Deze informatie helpt op zijn beurt winkels productinventaris beheren. 

- [Deep learning voor audio](https://blogs.technet.microsoft.com/machinelearning/2018/01/30/hearing-ai-getting-started-with-deep-learning-for-audio-on-azure/) in deze zelfstudie leert u hoe u met het trainen van een deep learning-model voor detectie audio gebeurtenis op de [stedelijke geluiden gegevensset](https://serv.cusp.nyu.edu/projects/urbansounddataset/urbansound8k.html) en biedt een overzicht van hoe u met audiogegevens werkt.

- [Classificatie van documenten](https://github.com/anargyri/lstm_han): In dit scenario ziet u hoe u bouwt en training van twee verschillende neural network-architecturen: HiÃ«rarchische aandacht netwerk- en Long korte termijn geheugen (LSTM) netwerk. Deze neurale netwerken voor het gebruik van de Keras-API voor deep learning voor het classificeren van documenten. Keras is een front-end aan drie van de meest populaire deep learning-frameworks: Microsoft Cognitive Toolkit, TensorFlow en Theano.
